# Quadrature Coupler Design using HFSS

## Overview

This project presents the design and simulation of a Microstrip Quadrature Coupler (3 dB Hybrid Coupler) using ANSYS HFSS 2025 R2 Student Version. The coupler is designed to equally divide the input power between output ports with a 90° phase difference, making it suitable for RF and microwave communication systems.

## Software Used

- ANSYS HFSS 2025 R2 Student

## Coupler Specifications

| Parameter | Value |
|------------|------------|
| Device Type | Quadrature Coupler |
| Technology | Microstrip |
| Center Frequency | 2.42 GHz |
| Return Loss (S11) | -59.65 dB |
| Coupling (S31) | -43.10 dB |
| Transmission (S21) | -28.87 dB |
| Isolation (S41) | -28.38 dB |
| VSWR | ≈ 1.08 |
